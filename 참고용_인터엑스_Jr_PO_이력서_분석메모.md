## 2. 데블록 경력 정리

기간: 2025.04 - 2026.05  
직무: 서비스 기획자 / PM

현재 PDF의 `3개 프로젝트 전담 기획 및 PM`은 아래처럼 바꾼다.

> 이레링크 외주개발 프로젝트와 내부 프로젝트 시픈의 기획/PM을 담당했습니다. 레퍼런스가 없는 러프한 요구사항을 바탕으로 레퍼런스 수집, 서비스 구조화, IA/User Flow, 기능 요구사항 정의, 개발 매니징, 일부 디자인을 수행했습니다.

### 이레링크

포지션: 데블록 실무 경력의 메인 사례
기간: 2025.05 - 2026.04

- 외국인 의료관광객, 병원, 통역사, 운영관리자가 얽힌 B2B 운영 시스템
- 클라이언트가 레퍼런스 없이 러프한 요구사항만 전달한 외주개발 프로젝트
- 본인이 레퍼런스 수집부터 기획, 개발 매니징, 일부 디자인까지 진행
- 산출물/범위: PRD, IA, 화면설계서, Figma 일부 디자인, QA 시트
- 웹/앱, Admin 등 운영 시스템 구조 기획
- 예약/매칭, 매출분석, 차트관리, 상품판매, 인센티브 정산, 데이터 대시보드, 웹빌더 기능으로 정리 가능
- 엑셀/칠판 중심 업무를 디지털 운영 시스템으로 전환한 경험으로 쓰면 좋다.
- 기존 매출 데이터 오류 발견은 데이터 정합성/운영 QA 근거로 좋다.
- 클라이언트가 업무 시간 단축에 큰 도움이 되었다고 피드백했다. 객관 수치는 없으므로 수치를 만들지 말고 정성 성과로 쓴다.

이력서 후보 문장:

- 레퍼런스가 없는 러프한 요구사항을 바탕으로 외국인 의료관광 B2B 운영 시스템의 서비스 구조, IA/User Flow, 주요 기능 요구사항을 정의했습니다.
- 웹/앱과 Admin 화면을 중심으로 예약, 매칭, 정산, 매출분석, 차트관리, 상품판매, 웹빌더 기능을 운영자 관점의 제품 구조로 정리했습니다.
- 외주개발 프로젝트에서 레퍼런스 수집, 기획, 개발 매니징, 일부 디자인을 담당하며 클라이언트 요구사항과 개발 구현 범위를 조율했습니다.

### 시픈

포지션: 데블록 내부 프로젝트, 테이스팃 아이디어의 제품화
기간: 2026.01 - 2026.04

- 이레링크 진행 중 본인이 내부 프로젝트로 제안
- 대학 시절 테이스팃에서 검증했던 아이디어가 그대로 살아 시픈으로 발전
- AI 취향분석/큐레이션, 리뷰/데이터 구조화, B2B 운영지원 플랫폼 맥락으로 정리 가능
- 본인 아이디어가 회사 내부 프로젝트로 채택/발전했다는 점이 강점
- 담당 범위: 아이디어 제안, 기획, 일부 디자인, 개발 매니징, GA4 작업

이력서 후보 문장:

- 사용자 취향/리뷰 데이터를 구조화해 큐레이션과 운영지원 기능으로 연결하는 서비스 방향을 기획했습니다.
- 초기 아이디어, 문제 정의, 서비스 방향, 주요 기능 구조를 제안해 내부 프로젝트로 발전시키는 데 기여했습니다.

## 3. 퇴사 후 단독 프로젝트

기간: 2026.04 - 2026.06

Dimetric, 스톡스토커, 사주중독은 데블록 프로젝트가 아니라 퇴사 후 혼자 만든 프로젝트로 정리한다. 스드맵은 웹 포트폴리오의 보조 사례로만 활용한다.

### 스톡스토커

포지션: 초기 사용 지표가 있는 데이터 제품 사례

- 국내/미국 주식 데이터 분석/스코어링/비교 서비스
- 100% 단독 기획, 개발, 배포
- 배포: https://stock-khaki.vercel.app
- GitHub: https://github.com/LAF-labs/stock
- 배포 후 1주 동안 약 1,000명 방문, 12,000건 종목 조회
- 피드백 기반으로 SEC/DART 공시 데이터 실시간 제공 및 룰베이스 요약 기능 업데이트
- 핵심 구조: KIS/yfinance, Supabase snapshot cache, 서버 메모리 캐시, partial response, refresh cooldown, 상세/비교 UX
- 인터엑스 연결점: 여러 출처의 데이터를 표준화하고, 신뢰도와 커버리지가 다른 데이터를 사용자가 판단 가능한 UI로 바꾸는 경험

이력서 후보 문장:

- 국내/미국 주식 데이터를 수집, 정규화, 캐싱, 스코어링해 종목 상세 분석과 최대 5개 종목 비교를 제공하는 서비스를 단독 기획/개발/배포했습니다.
- API 호출 제한, 데이터 커버리지 차이, 응답 지연을 고려해 snapshot cache, 서버 메모리 캐시, partial response, refresh cooldown 구조를 설계했습니다.
- ChatGPT, Claude, Codex를 활용해 요구사항 분해, 구현, 오류 분석, 테스트, 문서화를 반복하며 AI Native 작업 루프를 구축했습니다.

### Dimetric

포지션: AI Native 제품 구조와 제조/CAD 인접성을 보여주는 단독 프로젝트

- AI 기반 3D CAD/프린팅 모델 편집 도구
- GitHub: https://github.com/LAF-labs/dimetric
- 스크린샷:
  - `assets/dimetric-editor-gallery.png`: 채팅 패널, 3D 캔버스, 어셈블리/파트 패치 패널
  - `assets/dimetric-editor-canvas.png`: 채팅 패널과 3D 캔버스 중심 화면
- 문제 발견: 대학/지자체 3D 프린터 운용률이 5% 미만이고, 도안 제작(CAD) 허들이 높아 실제 활용이 어렵다는 점
- 로그인 후 프로젝트 보관함에서 프로젝트 생성, 열기, 삭제 가능
- 프로젝트별 미리보기, 이름, 수정일, 스냅샷 개수 제공
- 편집 화면은 왼쪽 채팅 패널, 중앙 Three.js 3D 캔버스, 오른쪽 수치/기능 패널 구조
- 사용자는 채팅으로 모델을 생성/수정하고, AI 권한 모드로 AI가 변경 가능한 파라미터 범위를 제어
- AI 생성 결과를 바로 적용하지 않고 구조화된 Graph Patch로 먼저 검증
- CAD 컴파일 검증 통과 후 모델 갱신 및 스냅샷 생성
- 잘못된 AI 응답이나 실패한 CAD 변경이 원본 프로젝트를 망가뜨리지 않도록 설계
- Python, build123d 등을 CAD engine으로 활용
- CAD 워커 기반 STL, 3MF, STEP, GLB 내보내기
- 최소 두께, 베드 크기, 오버행 등 3D 프린팅 위험 요소를 한국어 경고로 안내
- 인터엑스 연결점: AI 결과를 실제 제품 변경으로 연결할 때 필요한 권한 제어, 검증, 실패 방지, 3D/CAD 도메인 구조화 경험

이력서 후보 문장:

- AI 기반 3D CAD/프린팅 모델 편집 도구 Dimetric을 단독 기획/개발했습니다.
- 채팅 기반 모델 생성/수정, AI 권한 모드, Three.js 3D 캔버스, 수치/기능 패널을 결합한 편집 화면을 설계했습니다.
- AI 응답을 즉시 반영하지 않고 Graph Patch 검증과 CAD 컴파일 검증을 거친 뒤 모델을 갱신하고 스냅샷을 생성하도록 설계했습니다.
- CAD 워커를 통해 STL, 3MF, STEP, GLB 내보내기를 지원하고, 최소 두께/베드 크기/오버행 등 출력 위험 요소를 한국어 경고로 안내했습니다.

## 4. 대학 창업 프로젝트

대학 창업 프로젝트는 경력처럼 길게 펼치지 말고 `창업 프로젝트` 섹션으로 빼서 성과만 압축한다.

### 자투리

포지션: 대학 창업 프로젝트, 메인 기획자 겸 PM

- 대전/충청 인테리어 업체 81개 방문 설문
- IA, 플로우차트, 정책문서 작성
- Figma 전체 앱 디자인
- 외주 개발사 커뮤니케이션, Android MVP 앱 배포
- 80여 개 인테리어 업체 PoC 참여
- 2023년 4분기 매출 1,057만 원
- 거래건수 30.4%p 증가, 배송 서비스 이용률 11.3%p 증가
- 에코스타트업 4,628만 원, 청년창업사관학교 6,000만 원 선정

이력서 후보 문장:

- 대학 창업 프로젝트 자투리의 메인 기획자 겸 PM으로 시장조사, IA/정책문서, 앱 기획, 외주 개발 커뮤니케이션, MVP 배포를 담당했습니다.
- 80여 개 인테리어 업체 PoC와 Android MVP 운영을 통해 2023년 4분기 매출 1,057만 원을 만들고, 청년창업사관학교에 선정되었습니다.

### 테이스팃

포지션: 대학 창업 프로젝트, 팀 대표

- 팀 대표로 진행
- 매니아층 700명 설문, 자영업자 6명 인터뷰
- 800명 설문, 20명 심층 인터뷰
- 매니아층 구매 의향 82%
- 긍정적 제휴 검토 업체 7개, 그중 3개 확정
- 테이스팅 노트 MVP 테스트 600여 명
- 예비창업패키지 일반전형 서류 합격
- 제 2회 대한민국 대학 창업대전에서 한남대학교 대표팀으로 참가
- 경주 APEC 행사장 발표 경험

이력서 후보 문장:

- 대학 창업 프로젝트 테이스팃의 팀 대표로 수입주류 취향/리뷰 데이터 기반 큐레이션 서비스의 문제 정의, 시장검증, MVP 테스트를 주도했습니다.
- 700~800명 규모 설문과 20명 심층 인터뷰, 600여 명 MVP 테스트를 통해 사용자 니즈와 제품 방향을 검증했습니다.
- 제 2회 대한민국 대학 창업대전에서 한남대학교 대표팀으로 참가하는 등 대외 성과를 만들었습니다.
